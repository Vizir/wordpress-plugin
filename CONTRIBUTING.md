# Contributing to Vizir WordPress Plugin Boilerplate

:tada: First of all, thanks for your interest into contribute! :tada:

The target to **Vizir WordPress Plugin Boilerplate** is to be a very lightweight and simple boilerplate. Because that we will not create new features here, but we want to create a VZWPPB's Generator to make everyone's life better.

If you want to contribute with this generator, [let's talk](https://github.com/Vizir/wordpress-plugin-boilerplate/issues/1).

## Commiting ##

* The world talk english. Please keep this in mind when contributing.
* Write the commit's title and if necessary a complete message.

````
[Tag] Commit Title

The message if necessary...
````

### Tags ###

* **Feat:** A new Feature
* **Fix:** A bug fix
* **Style:** Changes that do not affect the meaning of the code (white-space, formatting, missing semi-colons, etc)
* **Refactor:** A bug fix
* **Docs:** Documentation only changes
